O conhecimento e as habilidades de um profissional full stack.

O teste é dividido em três partes:

múltipla escolha;

questões abertas

e

um exercício prático.

Teste de Avaliação para

Profissional Full Stack

Parte 1: Múltipla Escolha Qual das seguintes tecnologias é frequentemente usada no desenvolvimento front-end? a) Node.js b) Angular c) Express d) MongoDB

O que é um RESTful API? a) Um protocolo de segurança b) Um padrão arquitetural para construção de APIs c) Uma biblioteca de front-end d) Um tipo de banco de dados

Qual linguagem é mais comum para desenvolvimento back-end?

a) HTML b) CSS c) JavaScript d) SQL

Qual das seguintes opções é um gerenciador de pacotes para Node.js?

a) NPM

b) Composer

c) Bundler

d) Pip

O que significa "noSQL"? a) Um tipo de linguagem de programação b) Um banco de dados que não usa SQL como linguagem principal c) Uma abordagem de desenvolvimento ágil d) Um método de codificação de segurança.

Parte 2: Questões Abertas Explique a diferença entre bancos de dados relacionais e não relacionais. Dê exemplos de cada.

Descreva o que é o ciclo de vida de uma requisição HTTP em uma aplicação web.

Quais são algumas práticas recomendadas para garantir a segurança em uma aplicação full stack? Parte 3: Exercício Prático Desafio: Crie uma aplicação simples que permita aos usuários adicionar e listar tarefas (to-do list). A aplicação deve incluir: Um front-end em HTML/CSS/JavaScript. Um back-end simples usando Node.js e Express. Um banco de dados para armazenar as tarefas (você pode usar um banco de dados relacional ou não relacional, como SQLite ou MongoDB). Instruções: A aplicação deve ter uma interface básica onde o usuário pode adicionar uma nova tarefa e visualizar todas as tarefas existentes. O back-end deve expor uma API para manipulação das tarefas (criação e leitura). Critérios de Avaliação Múltipla Escolha: Cada questão correta vale 1 ponto. Questões Abertas: Avaliar a clareza, a precisão e a profundidade da resposta (0 a 5 pontos por resposta). Exercício Prático: Avaliar a funcionalidade, a estrutura do código, a organização e a estética da aplicação (0 a 10 pontos). Total de Pontos: 30 Esse teste ajudará a avaliar tanto o conhecimento teórico quanto as habilidades práticas de um profissional full stack. Se precisar de ajustes ou mais detalhes, é só avisar!
